Bienvenue sur Legion

Legion est une application de suivi de cohorte en lycée. Elle se base sur les données académiques et fournit un éventail de statistiques.

Première utilisation

La première chose à faire est de déclarer dans le fichier config.cfg la liste des sections classée par filière : GT (Générale et Technologique), Pro(fessionnelle) et Enseignement supérieur.

La seconde est d'importer la BEE (Base Élève Établissement) depuis le portail pagriates. Cette importation sera à refaire dans l'année pour mettre à jour les données élèves. Voir le tutoriel détaillé à ce sujet.

La troisième est d'associer chaque classe à son niveau et sa section, dans l'onglet Option.

Nouvelle année scolaire

Vous voulez utiliser les données d'une année précédente pour une nouvelle année scolaire ? Il suffit simplement de suivre la même démarche que pour la première utilisation ; les données seront automatiquement fusionnées.
