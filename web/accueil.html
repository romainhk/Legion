Bienvenue sur Legion

Legion est une application de suivi de cohorte en lycée. Elle se base sur les données académiques et fournit un outil de recherche et un panel de statistiques.

Première utilisation

Pour une explication détaillée, voir ce tutoriel. Dans les grandes lignes, après l'installation, il faut :

  • Déclarer dans le fichier config.cfg la liste des sections classée par filière : Générale, Technologique, Professionnelle et Enseignement supérieur.
  • Importer la BEE (Base Élève Établissement) depuis le portail pagriates. Cette importation sera à refaire dans l'année pour mettre à jour les données élèves.
  • Associer chaque classe à son niveau et sa section, dans l'onglet Options.

Nouvelle année scolaire

Vous voulez utiliser les données d'une année précédente pour une nouvelle année scolaire ? Il suffit simplement de suivre la même démarche que pour la première utilisation ; les données seront automatiquement fusionnées.
